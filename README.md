## ▶️ How to Run Locally

1. **Download or clone** the repository:
   ```bash
   git clone https://github.com/asifn0093/asif-nawaz-portfolio.git
   ```

2. **Open the folder:**
   ```bash
   cd asif-nawaz-portfolio
   ```

3. **Open `index.html`** in your browser — that's it! No installation needed.

## 🌐 Deploy on GitHub Pages

Follow these steps to get your **free live link**:

1. Go to your repository on GitHub
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source** → select **Deploy from a branch**
5. Under **Branch** → select `main` → folder `/root`
6. Click **Save**
7. Wait 1–2 minutes → your live link will be:

```
https://asifn0093.github.io/asif-nawaz-portfolio
```
